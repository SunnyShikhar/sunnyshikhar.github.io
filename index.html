Intramural Scheduling System

Developed a scheduling system for intramural sports at the University of Waterloo.

To improve the current scheduling procedure at uWaterloo, Faisal El Hussein and I designed a linear program to optimize the game schedule based on the particpant's preferences. The model chosen was a Binary Integer Program, where the students could pick a preference from 1-7 representing days of the week. Based on the available time slots, the program would output the optimal schedule.

Stock Decision Support System

Determine which stocks to purchase today and sell by closing to maximize profit, using neural networks.

The user specifies which stocks they're interested in purchasing, also the minimum and maximum number of stocks, and their budget. Microsoft Excel the retrieves the .csv file from Google Finance for each stock's pricing for the last 365 days, using VBA. These input values are then fed into a Artifical Neural Network where the proper weights are calculated to determine the closing price. The recommendations are made based on these closing prices.

Currently working on developing a web version using Python with the same functionality as the Excel file.

  • Date: April 2016
  • Category: Machine Learning

Bacon Numbers

Generated a graph data structure using Internet Movie Database (IMDB) to determine Degrees of Separation.

Used a graph data structure to determine degrees of separation of actors from actor Kevin Bacon. Used Shortest Path Alrogirthms such as depth-first search to find connections to Kevin Bacon and breadth-first search to find degrees of separation between actors.

  • Date: January 2015
  • Category: Algorithms and Data Structures
